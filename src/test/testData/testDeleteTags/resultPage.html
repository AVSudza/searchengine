Java | Чтение и запись текстовых файлов METANIT.COM Сайт о программировании Чтение и запись текстовых файлов Последнее обновление: 26.04.2018 Хотя с помощью ранее рассмотренных классов можно записывать текст в файлы, однако они предназначены прежде всего для работы с бинарными потоками данных, и их возможностей для полноценной работы с текстовыми файлами недостаточно. И для этой цели служат совсем другие классы, которые являются наследниками абстрактных классов Reader и Writer. Чтение файлов. Класс FileReader Класс FileReader наследуется от абстрактного класса Reader и предоставляет функциональность для чтения текстовых файлов. Для создания объекта FileReader мы можем использовать один из его конструкторов: Поскольку считанная порция файла может быть меньше 256 символов (например, в файле всего 73 символа), и если количество считанных данных меньше размера буфера (256), то выполняем копирование массива с помощью метода Arrays.copy. То есть фактически обрезаем массив buf, оставляя в нем только те символы, которые считаны из файла. Copyright metanit.com, 2024. Все права защищены.
